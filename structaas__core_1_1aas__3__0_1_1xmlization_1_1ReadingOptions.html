bool aas_core::aas_3_0::xmlization::ReadingOptions::additional_attributes = false

No XML attributes are expected in XML elements. Usually, attributes are considered errors and reported as such. However, some implementations add their own custom attributes, and we sometimes still want to parse such XML. If additional_attributes is set, the unexpected XML attributes will be ignored during parsing, and not reported.
