Detailed Description

A submodel element is an element suitable for the description and differentiation of assets.

It is recommended to add a IHasSemantics::semantic_id to a submodel element.

Constraint AASd-129:
If any IQualifier::kind value of ISubmodelElement::qualifiers (attribute qualifier inherited via Qualifiable) is equal to QualifierKind::kTemplateQualifier then the submodel element shall be part of a submodel template, i.e. a Submodel with ISubmodel::kind (attribute kind inherited via IHasKind) value is equal to ModellingKind::kTemplate.
