Detailed Description

Customize how instances should be serialized to XML.

We selected the defaults so that they can be used when you serialize to a file.

Usually, you want to write the namespace at the root element, and no prefixes are written in the XML names. However, if you are embedding the XML in a larger XML structure, you specify the namespace aliases and then use them as XML name prefixes. The prefix usually ends with a full colon (:).

We can not imagine in what situation you would want to write both the namespace and the prefix. Nevertheless, we allow for that possibility and do not throw any exception if you specify the both.
